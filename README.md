# RL-RoboCar
RoboCar Simulation for Reinforcement Learning
(carmunk.py has been modified and fixed for using pymunk 6.0.0)

## Install Pygame & Pymunk

`pip3 install pygame` (v2.0.1)

`pip3 install pymunk` (v6.0.0)

## Training
`python3 learning.py`

## Playing 
`python3 playing.py`

## Plotting
`python3 plotting.py`
